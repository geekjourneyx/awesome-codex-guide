# Security Policy

This repository curates Codex resources that may involve local files, shell commands, tokens, MCP servers, plugins, and other sensitive capabilities.

## Report A Problem

Open a GitHub issue if a listed resource:

- asks users to share API keys or tokens;
- bypasses official authentication;
- hides network, file-system, browser, database, or cloud access;
- ships a one-click script without auditable source;
- gives instructions that can leak secrets or damage a repository.

Do not paste secrets, tokens, private logs, or proprietary code in public issues.

## Curation Rule

Resources involving authentication, local file access, MCP, plugins, network automation, or cloud credentials must include a risk label and a short permission note before being recommended.
