# Codex 常见错误排查

## 401 / 认证失败

优先检查：

- 是否使用官方支持的认证方式。
- API key 或登录态是否有效。
- 配置文件路径是否正确。
- 终端是否重新加载了环境变量。
- 是否混用了第三方代理或旧配置。

不要把密钥贴到 issue、聊天窗口或第三方网页。

## 沙箱或权限失败

常见原因：

- Codex 试图写入项目外路径。
- 命令需要网络访问。
- 命令需要修改 Git refs 或索引。
- 脚本要访问系统级目录。

处理方式：先确认命令意图，再按最小权限授权。

## 配置不生效

检查：

- 配置文件路径。
- 文件格式。
- 环境变量优先级。
- 是否重启终端。
- 当前目录是否是预期项目。
