# Codex Troubleshooting

## 401 or authentication errors

Check official authentication first, then verify configuration paths, environment variables, and stale proxy settings. Never paste API keys into issues, chats, or third-party pages.

## Sandbox or permission errors

Common causes:

- Writing outside the project.
- Network access.
- Git index or ref changes.
- System-level file access.

Approve only commands you understand and keep the permission scope minimal.
