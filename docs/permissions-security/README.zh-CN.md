# 权限与安全

Codex 能读文件、改文件、运行命令。权限给得太大，出问题也会很快。

## 先守住这几条

- 只在可回滚的 Git 仓库中授权修改。
- 不要让 Codex 读取、打印、上传密钥。
- 网络访问、项目外写入、认证操作、部署操作必须单独确认。
- MCP、插件和子代理必须说明它们能访问什么。

## 看到这些要停一下

- 要求复制 API key 到第三方网页。
- 要求执行未审计的一键脚本。
- 要求关闭沙箱或绕过审批。
- 要求访问浏览器、数据库、云账号、GitHub token，但没有权限说明。

## 本仓库怎么处理

任何资源涉及下列能力，必须标注 `Security Review Needed`：

- API key 或 token。
- 本地文件系统写入。
- 网络请求。
- 浏览器、数据库、云服务、企业系统。
- 自动执行命令。
